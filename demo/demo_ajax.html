This example uses JavaScript, Ajax, and the DOM (Domain Object Model):

  • The JavaScript function showHint is called on each onkeyup event, i.e. each time a key has been pressed. This in turn calls https://www.w3schools.com/php/gethint.php?q=... on the server to get a suggestion.
  • The DOM is then used update the page with the retrieved suggestion. To be precise, the element labelled as textHint on the webpage is updated.
